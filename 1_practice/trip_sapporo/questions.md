# 📋 일정 작성 전 가족별 질문 목록

> 이 질문들에 대한 답변을 받아야 삿포로 5일 맞춤 일정 작성 가능

---

## 👨 아빠에게

1. **이동 수단**: 삿포로 현지에서 렌터카 vs 지하철·버스 대중교통?
2. **술/식사 스타일**: 시장에서 사서 숙소에서 vs 이자카야·게 전문점 식당?
3. **특식 예산**: 1인당 얼마 정도? (1만엔대 / 2만엔 이상 / 상관없음)
4. **꼭 가고 싶은 곳**: 삿포로 맥주박물관 / 오타루 운하 / 노보리베쓰 온천 / 후라노·비에이 중?
5. **숙소 형태**: 삿포로 시내 호텔 vs 온천 료칸 1박 포함?

## 👩 엄마에게

1. **카페 분위기**: 북유럽 감성 / 일본 레트로 / 뷰 맛집 / 디저트 특화?
2. **쇼핑 우선순위**: 뷰티·화장품 / 과자·기념품 / 패션 / 아이 장난감?
3. **온천 선호도**: 료칸 온천 vs 당일치기 온천 vs 굳이 안 해도 됨?
4. **꼭 찍고 싶은 포인트**: 벚꽃 / 홋카이도대학 가로수길 / 오타루 운하 야경 / 양 목장?

---

## 👧 수아 (아빠·엄마가 대신 답변)

1. **먹고 싶은 것**: 라멘 / 초밥 / 소프트크림 / 돈가스 / 카레 중 특별히?
2. **동물 체험 관심도**: 양 목장·버터 만들기 체험 / 동물원 / 수족관?
3. **눈 vs 봄꽃**: 잔설 밟기·눈 관련 활동 vs 봄꽃·자연 산책 어느 쪽?
4. **실내 체험**: 과학관 / 수족관 / 눈·얼음 테마 체험관 중 흥미 있는 곳?
